LaTeX with Sublime Text

Apr 1st 2015

Why Sublime Text

In most LaTeX tutorials, the author will recommend you to use WinEdt and TeXMake. These are good choice, but as developers, we already have dozens of tools for text editor. We don't want another one just for LaTeX. Can't we just do it with our most frequently used tools ? And that is where Sublime Text comes in.

Front-end developers can't be too familiar with Sublime Text. It is THE AWESOME TOOL that 9 out of 10 developers are using it. Its popularity and extendibility, even when comparing with Eclipse, shows no disadvantage. Sublime Text had syntax highlight for LaTeX built in. But it doesn't contain TeX, which means it can't compile. So we need a little configuration before we start to use it. Here I'm gonna demostrate how to do it on Windows platform. (If your are using OSX or Linux, follow this article: https://github.com/SublimeText/LaTeXTools/blob/master/README.markdown)

Preparation

Before we start, we need to download all the gears we need and install them.

MiKTeX An up-to-date implementation of TeX/LaTeX and related programs for Windows Download
Sublime Text The text editor you'll fall in love with. Perfect for Front-end Development Download
LaTeX Tools A Sublime Text Plugin for writing LaTeX. Package Control is strongly recommended to install this package. If you need an offline package. Click the link on the right. Download
Sumatra PDF A free PDF reader, LaTeXTools by default use it to preview the created PDF file. Download

Configure LaTeXTools

Open Sublime Text. Press Ctrl+Shift+P to pull out Package Control. Search for "LaTeXTools: Reconfigure and migrate settings" and hit enter. And it's done. This command creates a configuration file named "LaTeXTools.sublime-settings" in your "User" directory.

Configure SumatraPDF

Step 1: add the SumatraPDF directory to your PATH, This is important, or the next step won't work.

Step 2: Open up a command-line console (cmd.exe) and type in the following code:(change the install path into yours)

sumatrapdf.exe -inverse-search "\"C:\Program Files\Sublime Text 3\sublime_text.exe\" \"%f:%l\""

Usage

By now, the configuration is done. WE can start writing LaTeX with Sublime Text. Every time you done editing a tex file. Save it (If you just created a new tex file, you must save it before you build it. Or it won't know what file this is). Press Ctrl+B. Sublime Text would build the file using LaTeXTools. When it's done. SumatraPDF will open up automatically and let you preview your work. The preview will auto-update itself after a new build, all you need to do is press Ctrl+B.

Summary

Now we can use our favourite text editor to write LaTeX. Perfect isn't it ?
